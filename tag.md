### Basic

| Tag          | Description                              |
| :----------- | :--------------------------------------- |
| `<!DOCTYPE>` | Defines the document type                |
| `<html>`     | Defines an HTML document                 |
| `<title>`    | Defines a title for document             |
| `<body>`     | Defines the document's body              |
| `<hn>`       | Defines HTML headings(`<h1> to <h6>`)    |
| `<p>`        | Defines a paragraph                      |
| `<br>`       | Inserts a single line break              |
| `<hr>`       | Defines a thematic change in the content |
| `<!--...-->` | Defines a comment                        |

### Formatting

| Tag            | Description                                                                                |
| :------------- | :----------------------------------------------------------------------------------------- |
| `<abbr>`       | Defines an abbreviation or an acronym                                                      |
| `<address>`    | Defines contact information for the author/owner of a document/article                     |
| `<pre>`        | Defines preformatted text                                                                  |
| `<small>`      | Defines smaller text                                                                       |
| `<time>`       | Defines a date/time                                                                        |
| `<mark>`       | Defines marked/highlighted text                                                            |
| `<wbr>`        | Defines a possible line-break                                                              |
| `<bdi>`        | Isolates a parts of text that might be formatted in a direction from other text outside it |
| `<bdo>`        | Overrides the current text direction                                                       |
| `<blockquote>` | Defines a section that is quoted from another source                                       |
| `<q>`          | Defines a short quotation                                                                  |
| `<rp>`         | Defines what to show in browsers that do not support annotations                           |
| `<rt>`         | Defines an explanation/pronunciation of characters(for East Asian typography)              |
| `<ruby>`       | Defines a ruby annotation(for East Asian typography)                                       |
| `<meter>`      | Defines a scalar measurement within a known range(a gauge)                                 |
| `<progress>`   | Represents the progress of a task                                                          |
| `<sub>`        | Defines subscripted text                                                                   |
| `<sup>`        | Defines superscripted text                                                                 |

### Font

| Tag        | Description                                                          | Default CSS             |
| :--------- | :------------------------------------------------------------------- | :---------------------- |
| `<b>`      | Defines bold text                                                    | font-weight:bold        |
| `<strong>` | Defines important text                                               | font-weight:bold        |
| `<u>`      | Defines text that should be stylistically different from normal text | font-style:underline    |
| `<ins>`    | Defines a text that has been inserted into a document                | font-style:underline    |
| `<s>`      | Defines text that is no longer correct                               | font-style:line-through |
| `<del>`    | Defines text that has been deleted from a document                   | font-style:line-through |
| `<i>`      | Defines a part of text in an alternate voice or mood                 | font-style:italic       |
| `<em>`     | Defines emphasized text                                              | font-style:italic       |
| `<dfn>`    | Represents the defining instance of a term                           | font-style:italic       |
| `<var>`    | Defines a variable                                                   | font-style:italic       |
| `<cite>`   | Defines the title of a work                                          | font-style:italic       |
| `<kbd>`    | Defines keyboard input                                               | font-family:monospace   |
| `<code>`   | Defines a piece of computer code                                     | font-family:monospace   |
| `<samp>`   | Defines sample output from a computer program                        | font-family:monospace   |
